# Changelog

## 0.1.0 - 2026-06-08

首个可长期自用的科研日程驾驶舱版本。

- 建立今日、周计划、项目、工作台、复盘和设置六个核心视图。
- 完成全局搜索、状态过滤、今日/本周快捷跳转。
- 支持任务新增、编辑、复制、删除、状态流转、批量操作和撤销。
- 增强今日排序、周负载、超载提醒和时间块可视化。
- 增加项目详情、里程碑、风险提示、进度趋势和项目内创建任务。
- 增加复盘统计、自动建议、手写复盘、本周结论和下周调整。
- 增加 localStorage 持久化、版本校验、JSON/CSV/Markdown 导出、导入恢复和备份。
- 增加分类、项目、字段模板、视图预设、复盘指标和显示密度自定义。
- 增加 Vitest/Testing Library 测试，覆盖 selector、持久化校验和核心任务流程。
- 增加 README、隐私说明、应用图标和 GitHub Pages 发布配置。
