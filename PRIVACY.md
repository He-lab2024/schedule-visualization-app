# Privacy

科研日程驾驶舱是本地优先的个人工具。

## 默认数据存储

- 任务、项目、分类、模板、视图配置和复盘笔记默认保存在当前浏览器的 `localStorage`。
- 应用本身不包含后端服务，也不会主动把数据上传到服务器。
- 如果你清除浏览器站点数据，应用内保存的数据也会被清除。
- 启用“本地数据文件夹”后，浏览器会通过 File System Access API 请求你明确选择文件夹。应用只会在授权文件夹内读写 `schedule-data.json` 和 `backups/` 下的备份文件，不会静默扫描或写入任意硬盘路径。

## 备份文件

- JSON、CSV 和 Markdown 导出文件由你自己保存和管理。
- 导出的 JSON 可能包含完整任务、项目、复盘笔记和个人工作节奏信息。
- 建议把备份文件放在可信位置，不要提交到公开仓库。

## 第三方服务

- 部署到 GitHub Pages 后，静态资源由 GitHub 托管。
- 应用运行时仍然只在浏览器本地保存业务数据。
- 如果未来接入同步、登录或云存储，应在本文件中更新对应说明。
